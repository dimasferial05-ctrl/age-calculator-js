# Age Calculator

Aplikasi kalkulator umur yang akurat menggunakan library **Luxon** untuk perhitungan tanggal dan **Flatpickr** untuk antarmuka pemilihan tanggal yang modern. Proyek ini dibangun sebagai solusi tantangan dari [roadmap.sh]([https://roadmap.sh/](https://roadmap.sh/projects/age-calculator)).

## ✨ Fitur Utama

- **Custom Datepicker:** Menggunakan library Flatpickr untuk pengalaman memilih tanggal yang lebih baik dan konsisten di semua browser.
- **Perhitungan Akurat:** Menggunakan Luxon untuk menghitung perbedaan waktu dalam satuan tahun dan bulan secara presisi, termasuk penanganan tahun kabisat.
- **Validasi Input:** Memastikan tanggal lahir telah dipilih dan mencegah pemilihan tanggal di masa depan (max date: today).
- **Desain Minimalis:** Antarmuka monokrom yang bersih, modern, dan responsif.
- **Integrasi Library Eksternal:** Implementasi library pihak ketiga melalui CDN untuk fungsionalitas yang kuat dan efisien.

## 📂 Struktur Proyek

- `index.html` - Struktur utama dan pemuatan library eksternal (Luxon & Flatpickr).
- `style.css` - Desain antarmuka monokrom yang responsif.
- `script.js` - Logika inisialisasi datepicker dan kalkulasi umur.

## 🛠️ Teknologi yang Digunakan

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **Libraries:** [Luxon](https://moment.github.io/luxon/), [Flatpickr](https://flatpickr.js.org/)
